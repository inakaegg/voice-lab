SkitVoice

かんたんスキット生成

台本と参照音声から、指定した声でセリフ音声を生成します。

管理

Script

台本

行頭に 1-4 / A-D を付けると話者を指定できます。話者ごとに対応する参照音声を指定してください。

Voices

参照音声

URLから取得

動画/音声URLの指定位置から参照音声を作成します。

Mode

生成モード

指定台詞モード
ON。話者ごとに1行生成し、ASRで台本順へ再配置します。
低スコア行の再生成
ON。ASRスコアが0.65未満の行を、台本行数から自動算出した上限まで再生成します。
